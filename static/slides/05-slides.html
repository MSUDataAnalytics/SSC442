Nomenclature reminder

We have pairs of data, (xi,yi)(x_i, y_i)(xi​,yi​), for i=1,2,…ni = 1, 2, \ldots ni=1,2,…n, where nnn is the sample size of the dataset.

We use iii as an index, which tells us the numbered entry of the each data pair. As discussed previously, xix_ixi​ is the predictor (explanatory) variable. The predictor variable is used to help predict or explain the response (target, outcome) variable, yiy_iyi​.

In the cars example, we are interested in using the predictor variable speed to predict and explain the response variable dist.

Broadly speaking, we would like to model the relationship between XXX and YYY using the form

Y=f(X)+ϵ.Y = f(X) + \epsilon.Y=f(X)+ϵ.

Target functions

The function fff describes the functional relationship between the two variables, and the ϵ\epsilonϵ term is used to account for error. This indicates that if we plug in a given value of XXX as input, our output is a value of YYY, within a certain range of error. You could think of this a number of ways:

  • Response = Prediction + Error
  • Response = Signal + Noise
  • Response = Model + Unexplained
  • Response = Deterministic + Random
  • Response = Explainable + Unexplainable

With this in mind, we would like to restrict our choice of f(X)f(X)f(X) to linear functions of XXX. We will write our model using β1\beta_1β1​ for the slope, and β0\beta_0β0​ for the intercept,

Y=β0+β1X+ϵ.Y = \beta_0 + \beta_1 X + \epsilon.Y=β0​+β1​X+ϵ.

Simple Linear Regression Model

We now define what we will call the simple linear regression model,

Yi=β0+β1xi+ϵiY_i = \beta_0 + \beta_1 x_i + \epsilon_iYi​=β0​+β1​xi​+ϵi​

where

ϵi∼N(0,σ2).\epsilon_i \sim N(0, \sigma^2).ϵi​∼N(0,σ2).

That is, the ϵi\epsilon_iϵi​ are independent and identically distributed (iid) normal random variables with mean 000 and variance σ2\sigma^2σ2. This model has three parameters to be estimated: β0\beta_0β0​, β1\beta_1β1​, and σ2\sigma^2σ2, which are fixed, but unknown constants.

We have slightly modified our notation here. We are now using YiY_iYi​ and xix_ixi​, since we will be fitting this model to a set of nnn data points, for i=1,2,…ni = 1, 2, \ldots ni=1,2,…n.

We use capital YYY to indicate a random variable, and lower case yyy to denote a potential value of the random variable. Since we will have nnn observations, we have nnn random variables YiY_iYi​ and their possible values yiy_iyi​.

In the simple linear regression model, the xix_ixi​ are assumed to be fixed, known constants, and are thus notated with a lower case variable. The response YiY_iYi​ remains a random variable because of the random behavior of the error variable, ϵi\epsilon_iϵi​. That is, each response YiY_iYi​ is tied to an observable xix_ixi​ and a random, unobservable, ϵi\epsilon_iϵi​.

Essentially, we could explicitly think of the YiY_iYi​ as having a different distribution for each XiX_iXi​. In other words, YiY_iYi​ has a conditional distribution dependent on the value of XiX_iXi​, written xix_ixi​. Doing so, we still make no distributional assumptions of the XiX_iXi​, since we are only interested in the distribution of the YiY_iYi​ for a particular value xix_ixi​.

Yi∣Xi∼N(β0+β1xi,σ2)Y_i \mid X_i \sim N(\beta_0 + \beta_1 x_i, \sigma^2)Yi​∣Xi​∼N(β0​+β1​xi​,σ2)

The random YiY_iYi​ are a function of xix_ixi​, thus we can write its mean as a function of xix_ixi​,

E[Yi∣Xi=xi]=β0+β1xi.\text{E}[Y_i \mid X_i = x_i] = \beta_0 + \beta_1 x_i.E[Yi​∣Xi​=xi​]=β0​+β1​xi​.

However, its variance remains constant for each xix_ixi​,

Var[Yi∣Xi=xi]=σ2.\text{Var}[Y_i \mid X_i = x_i] = \sigma^2.Var[Yi​∣Xi​=xi​]=σ2.

This is visually displayed in the image below. We see that for any value xxx, the expected value of YYY is β0+β1x\beta_0 + \beta_1 xβ0​+β1​x. At each value of xxx, YYY has the same variance σ2\sigma^2σ2.

Linear Regression Assumptions

Often, we directly talk about the assumptions that this model makes. They can be cleverly shortened to LINE.

  • Linear. The relationship between YYY and xxx is linear, of the form β0+β1x\beta_0 + \beta_1 xβ0​+β1​x.
  • Independent. The errors ϵ\epsilonϵ are independent.
  • Normal. The errors, ϵ\epsilonϵ are normally distributed. That is the "error" around the line follows a normal distribution.
  • Equal Variance. At each value of xxx, the variance of YYY is the same, σ2\sigma^2σ2.

We are also assuming that the values of xxx are fixed, that is, not random. We do not make a distributional assumption about the predictor variable.

As a side note, we will often refer to simple linear regression as SLR. Some explanation of the name SLR:

  • Simple refers to the fact that, for now, we are using a single predictor variable. This greatly simplifies the mathematics. Later we will use multiple predictor variables.
  • Linear tells us that our model for YYY is a linear combination of the predictors XXX. (In this case just the one.) Right now, this always results in a model that is a line, but later we will see how this is not always the case.
  • Regression simply means that we are attempting to measure the relationship between a response variable and (one or more) predictor variables. In the case of SLR, both the response and the predictor are numeric variables.

This approach takes YYY as a linear function of XXX. But how do we actually define a good line? There are an infinite number of lines we could use (using our notation from previous lectures, ∣H∣=∞|\mathcal{H}| = \infty∣H∣=∞), so we will attempt to find one with "small errors." That is a line with as many points as close to it as possible. The questions now becomes: how do we find such a line? There are many approaches we could take.

We could find the line that has the smallest maximum distance from any of the points to the line. That is,

argminβ0,β1max⁡∣yi−(β0+β1xi)∣.\underset{\beta_0, \beta_1}{\mathrm{argmin}} \max|y_i - (\beta_0 + \beta_1 x_i)|.β0​,β1​argmin​max∣yi​−(β0​+β1​xi​)∣.

We could find the line that minimizes the sum of all the distances from the points to the line. That is,

argminβ0,β1∑i=1n∣yi−(β0+β1xi)∣.\underset{\beta_0, \beta_1}{\mathrm{argmin}} \sum_{i = 1}^{n}|y_i - (\beta_0 + \beta_1 x_i)|.β0​,β1​argmin​i=1∑n​∣yi​−(β0​+β1​xi​)∣.

We could find the line that minimizes the sum of all the squared distances from the points to the line. That is,

argminβ0,β1∑i=1n(yi−(β0+β1xi))2.\underset{\beta_0, \beta_1}{\mathrm{argmin}} \sum_{i = 1}^{n}(y_i - (\beta_0 + \beta_1 x_i))^2.β0​,β1​argmin​i=1∑n​(yi​−(β0​+β1​xi​))2.

This last option is called the method of least squares.[2]

Least Squares Approach

Given observations (xi,yi)(x_i, y_i)(xi​,yi​), for i=1,2,…ni = 1, 2, \ldots ni=1,2,…n, we want to find values of β0\beta_0β0​ and β1\beta_1β1​ which minimize

f(β0,β1)=∑i=1n(yi−(β0+β1xi))2=∑i=1n(yi−β0−β1xi)2.f(\beta_0, \beta_1) = \sum_{i = 1}^{n}(y_i - (\beta_0 + \beta_1 x_i))^2 = \sum_{i = 1}^{n}(y_i - \beta_0 - \beta_1 x_i)^2.f(β0​,β1​)=i=1∑n​(yi​−(β0​+β1​xi​))2=i=1∑n​(yi​−β0​−β1​xi​)2.

We will call these values β^0\hat{\beta}_0β^​0​ and β^1\hat{\beta}_1β^​1​.

First, we take a partial derivative with respect to both β0\beta_0β0​ and β1\beta_1β1​.

Practical Implementation

Traditionally we would now calculate β^0\hat{\beta}_0β^​0​ and β^1\hat{\beta}_1β^​1​ by hand for the cars dataset. However because we are living in the 21st century and are intelligent (or lazy or efficient, depending on your perspective), we will utilize R to do the number crunching for us.

To keep some notation consistent with above mathematics, we will store the response variable as y and the predictor variable as x.

x = cars$speed
y = cars$dist

We then calculate the three sums of squares defined above.

Sxy = sum((x - mean(x)) * (y - mean(y)))
Sxx = sum((x - mean(x)) ^ 2)
Syy = sum((y - mean(y)) ^ 2)
c(Sxy, Sxx, Syy)

Then finally, we can calculate β^0\hat{\beta}_0β^​0​ and β^1\hat{\beta}_1β^​1​.

beta_1_hat = Sxy / Sxx
beta_0_hat = mean(y) - beta_1_hat * mean(x)
c(beta_0_hat, beta_1_hat)

What do these values tell us about our dataset?

The slope parameter β1\beta_1β1​ tells us that for an increase in speed of one mile per hour, the mean stopping distance increases by β1\beta_1β1​. It is important to specify that we are talking about the mean. Recall that β0+β1x\beta_0 + \beta_1 xβ0​+β1​x is the mean of YYY, in this case stopping distance, for a particular value of xxx. (In this case speed.) So β1\beta_1β1​ tells us how the mean of YYY is affected by a change in xxx.

Similarly, the estimate β^1\hat{\beta}_1β^​1​ tells us that for an increase in speed of one mile per hour, the estimated mean stopping distance increases by β^1\hat{\beta}_1β^​1​ feet. Here we should be sure to specify we are discussing an estimated quantity. Recall that y^\hat{y}y^​ is the estimated mean of YYY, so β^1\hat{\beta}_1β^​1​ tells us how the estimated mean of YYY is affected by changing xxx.

The intercept parameter β0\beta_0β0​ tells us the mean stopping distance for a car traveling zero miles per hour. (Not moving.) The estimate β^0\hat{\beta}_0β^​0​ tells us that the estimated mean stopping distance for a car traveling zero miles per hour is β^0\hat{\beta}_0β^​0​ feet.[3]

Making Predictions

We can now write the fitted or estimated line,

y^=β^0+β^1x.\hat{y} = \hat{\beta}_0 + \hat{\beta}_1 x.y^​=β^​0​+β^​1​x.

We can now use this line to make predictions. First, let's see the possible xxx values in the cars dataset. Since some xxx values may appear more than once, we use the unique() to return each unique value only once.

unique(cars$speed)

Let's make a prediction for the stopping distance of a car traveling at 8 miles per hour.

y^=β^0+β^1×8\hat{y} = \hat{\beta}_0 + \hat{\beta}_1 \times 8y^​=β^​0​+β^​1​×8

beta_0_hat + beta_1_hat * 8

8 %in% unique(cars$speed)
21 %in% unique(cars$speed)

min(cars$speed) < 21 & 21 < max(cars$speed)

Our prediction is simply given by
y^=β^0+β^1×21\hat{y} = \hat{\beta}_0 + \hat{\beta}_1 \times 21y^​=β^​0​+β^​1​×21
and is easily implemented by

beta_0_hat + beta_1_hat * 21

Lastly, we can make a prediction for the stopping distance of a car traveling at 100 miles per hour. This is considered extrapolation as 100 is not an observed value of xxx and is outside data range. We should be less confident in predictions of this type; we'll formalize this notion later.

Residuals

If we think of our model as "Response = Prediction + Error," we can then write it as

y=y^+e.y = \hat{y} + e.y=y^​+e.

We then define a residual to be the observed value minus the predicted value.

ei=yi−y^ie_i = y_i - \hat{y}_iei​=yi​−y^​i​

Variance Estimation

We'll now use the residuals for each of the points to create an estimate for the variance, σ2\sigma^2σ2.

Recall that,

E[Yi∣Xi=xi]=β0+β1xi.\text{E}[Y_i \mid X_i = x_i] = \beta_0 + \beta_1 x_i.E[Yi​∣Xi​=xi​]=β0​+β1​xi​.

So,

y^i=β^0+β^1xi\hat{y}_i = \hat{\beta}_0 + \hat{\beta}_1 x_iy^​i​=β^​0​+β^​1​xi​

is a natural estimate for the mean of YiY_iYi​ for a given value of xix_ixi​.

Also, recall that when we specified the model, we had three unknown parameters; β0\beta_0β0​, β1\beta_1β1​, and σ2\sigma^2σ2. The method of least squares gave us estimates for β0\beta_0β0​ and β1\beta_1β1​, however, we have yet to see an estimate for σ2\sigma^2σ2. We will now define se2s_e^2se2​ which will be an estimate for σ2\sigma^2σ2.

se2=1n−2∑i=1n(yi−(β^0+β^1xi))2=1n−2∑i=1n(yi−y^i)2=1n−2∑i=1nei2\begin{aligned} s_e^2 &= \frac{1}{n - 2} \sum_{i = 1}^{n}(y_i - (\hat{\beta}_0 + \hat{\beta}_1 x_i))^2 \\ &= \frac{1}{n - 2} \sum_{i = 1}^{n}(y_i - \hat{y}_i)^2 \\ &= \frac{1}{n - 2} \sum_{i = 1}^{n} e_i^2 \end{aligned}se2​​=n−21​i=1∑n​(yi​−(β^​0​+β^​1​xi​))2=n−21​i=1∑n​(yi​−y^​i​)2=n−21​i=1∑n​ei2​​

This probably seems like a natural estimate, aside from the use of n−2n - 2n−2, which we will ignore for now (there's a natural reason for this correction but it is unrelated to the broader themes in this course). It should actually look rather similar to something you have seen before.

Here, s2s^2s2 is the estimate of σ2\sigma^2σ2 when we have a single random variable XXX. In this case xˉ\bar{x}xˉ is an estimate of μ\muμ which is assumed to be the same for each xxx.

Now, in the regression case, with se2s_e^2se2​ each yyy has a different mean because of the relationship with xxx. Thus, for each yiy_iyi​, we use a different estimate of the mean, that is y^i\hat{y}_iy^​i​.

Just as with the univariate measure of variance, this value of r round(s2_e, 2) doesn't have a practical interpretation in terms of stopping distance. Taking the square root, however, computes the standard deviation of the residuals, also known as residual standard error.

s_e = sqrt(s2_e)
s_e

This tells us that our estimates of mean stopping distance are "typically" off by r round(s_e, 2) feet.

Decomposition of Variation

We can re-express yi−yˉy_i - \bar{y}yi​−yˉ​, which measures the deviation of an observation from the sample mean, in the following way,

yi−yˉ=(yi−y^i)+(y^i−yˉ).y_i - \bar{y} = (y_i - \hat{y}_i) + (\hat{y}_i - \bar{y}).yi​−yˉ​=(yi​−y^​i​)+(y^​i​−yˉ​).

This is the common mathematical trick of "adding zero." In this case we both added and subtracted y^i\hat{y}_iy^​i​.

Here, yi−y^iy_i - \hat{y}_iyi​−y^​i​ measures the deviation of an observation from the fitted regression line and y^i−yˉ\hat{y}_i - \bar{y}y^​i​−yˉ​ measures the deviation of the fitted regression line from the sample mean.

If we square then sum both sides of the equation above, we can obtain the following,

∑i=1n(yi−yˉ)2=∑i=1n(yi−y^i)2+∑i=1n(y^i−yˉ)2.\sum_{i=1}^{n}(y_i - \bar{y})^2 = \sum_{i=1}^{n}(y_i - \hat{y}_i)^2 + \sum_{i=1}^{n}(\hat{y}_i - \bar{y})^2.i=1∑n​(yi​−yˉ​)2=i=1∑n​(yi​−y^​i​)2+i=1∑n​(y^​i​−yˉ​)2.

The quantity "Sum of Squares Total," or SST\text{SST}SST, represents the total variation of the observed yyy values. This should be a familiar looking expression. Note that,

s2=1n−1∑i=1n(yi−yˉ)2=1n−1SST.s ^ 2 = \frac{1}{n - 1}\sum_{i=1}^{n}(y_i - \bar{y})^2 = \frac{1}{n - 1} \text{SST}.s2=n−11​i=1∑n​(yi​−yˉ​)2=n−11​SST.

The quantity "Sum of Squares Regression," SSReg\text{SSReg}SSReg, represents the explained variation of the observed yyy values.

Residual Sum of Squares

The quantity RSS\text{RSS}RSS, or "Residual Sum of Squares," represents the unexplained variation of the observed yyy values.

SST   = sum((y - mean(y)) ^ 2)
SSReg = sum((y_hat - mean(y)) ^ 2)
SSE   = sum((y - y_hat) ^ 2)
c(SST = SST, SSReg = SSReg, SSE = SSE)

Note that,

se2=SSEn−2.s_e^2 = \frac{\text{SSE}}{n - 2}.se2​=n−2SSE​.

We can use R to verify that this matches our previous calculation of se2s_e^2se2​.

These three measures also do not have an important practical interpretation individually. But together, they're about to reveal a new statistic to help measure the strength of a SLR model.

Coefficient of Determination

The coefficient of determination, R2R^2R2, is defined as

R2=SSRegSST=∑i=1n(y^i−yˉ)2∑i=1n(yi−yˉ)2=SST−SSESST=1−SSESST=1−∑i=1n(yi−y^i)2∑i=1n(yi−yˉ)2=1−∑i=1nei2∑i=1n(yi−yˉ)2\begin{aligned} R^2 &= \frac{\text{SSReg}}{\text{SST}} = \frac{\sum_{i=1}^{n}(\hat{y}_i - \bar{y})^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2} \\[2.5ex] &= \frac{\text{SST} - \text{SSE}}{\text{SST}} = 1 - \frac{\text{SSE}}{\text{SST}} \\[2.5ex] &= 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2} = 1 - \frac{\sum_{i = 1}^{n}e_i^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2} \end{aligned}R2​=SSTSSReg​=∑i=1n​(yi​−yˉ​)2∑i=1n​(y^​i​−yˉ​)2​=SSTSST−SSE​=1−SSTSSE​=1−∑i=1n​(yi​−yˉ​)2∑i=1n​(yi​−y^​i​)2​=1−∑i=1n​(yi​−yˉ​)2∑i=1n​ei2​​​

The coefficient of determination is interpreted as the proportion of observed variation in yyy that can be explained by the simple linear regression model.

Simulating SLR

We return again to more examples of simulation. This will be a common theme!

In practice you will never have a true model, and you will use data to attempt to recover information about the unknown true model. With simulation, we decide the true model and simulate data from it. Then, we apply a method to the data, in this case least squares. Now, since we know the true model, we can assess how well it did.

For this example, we will simulate n=21n = 21n=21 observations from the model

Y=5−2x+ϵ.Y = 5 - 2 x + \epsilon.Y=5−2x+ϵ.

That is β0=5\beta_0 = 5β0​=5, β1=−2\beta_1 = -2β1​=−2, and let ϵ∼N(μ=0,σ2=9)\epsilon \sim N(\mu = 0, \sigma^2 = 9)ϵ∼N(μ=0,σ2=9). Or, even more succinctly we could write

Y∣X∼N(μ=5−2x,σ2=9).Y \mid X \sim N(\mu = 5 - 2 x, \sigma^ 2 = 9).Y∣X∼N(μ=5−2x,σ2=9).

We first set the true parameters of the model to be simulated.

num_obs = 21
beta_0  = 5
beta_1  = -2
sigma   = 3

Next, we obtain simulated values of ϵi\epsilon_iϵi​ after setting a seed for reproducibility.

set.seed(1)
epsilon = rnorm(n = num_obs, mean = 0, sd = sigma)

Now, since the xix_ixi​ values in SLR are considered fixed and known, we simply specify 20 values. Another common practice is to generate them from a uniform distribution, and then use them for the remainder of the analysis.

x_vals = seq(from = 0, to = 10, length.out = num_obs)
# set.seed(1)
# x_vals = runif(num_obs, 0, 10)

We then generate the yyy values according the specified functional relationship.

y_vals = beta_0 + beta_1 * x_vals + epsilon

The data, (xi,yi)(x_i, y_i)(xi​,yi​), represent a possible sample from the true distribution. Now to check how well the method of least squares works, we use lm() to fit the model to our simulated data, then take a look at the estimated coefficients.

The function below, sim_slr(), can be used for the same task as above, but is much more flexible. Notice that we provide x to the function, instead of generating x inside the function. In the SLR model, the xix_ixi​ are considered known values. That is, they are not random, so we do not assume a distribution for the xix_ixi​. Because of this, we will repeatedly use the same x values across all simulations.

sim_slr = function(x, beta_0 = 10, beta_1 = 5, sigma = 1) {
  n = length(x)
  epsilon = rnorm(n, mean = 0, sd = sigma)
  y = beta_0 + beta_1 * x + epsilon
  data.frame(predictor = x, response = y)
}
