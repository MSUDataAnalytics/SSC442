Introduction by Example

The last lecture we saw how to fit a model that assumed a linear relationship between a response variable and a single predictor variable. Specifically, we defined the simple linear regression model,

Yi=β0+β1xi+ϵiY_i = \beta_0 + \beta_1 x_i + \epsilon_iYi​=β0​+β1​xi​+ϵi​

where ϵi∼N(0,σ2)\epsilon_i \sim N(0, \sigma^2)ϵi​∼N(0,σ2).

However, it is rarely the case that we want to explore merely a single predictor variable. It is also rarely the case that a response variable will only depend on a single variable.[1] We will extend the linear model to allow a response to depend on multiple predictors. The code below loads a useful dataset.

We will once again discuss a dataset with information about cars. This dataset, which can be found at the UCI Machine Learning Repository contains a response variable mpg which stores the city fuel efficiency of cars, as well as several predictor variables for the attributes of the vehicles. We load the data, and perform some basic tidying before moving on to analysis.

For now we will focus on using two variables, wt and year, as predictor variables. That is, we would like to model the fuel efficiency (mpg) of a car as a function of its weight (wt) and model year (year). To do so, we will define the following linear model,

Yi=β0+β1xi1+β2xi2+ϵi,i=1,2,…,nY_i = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \epsilon_i, \qquad i = 1, 2, \ldots, nYi​=β0​+β1​xi1​+β2​xi2​+ϵi​,i=1,2,…,n

where ϵi∼N(0,σ2)\epsilon_i \sim N(0, \sigma^2)ϵi​∼N(0,σ2). In this notation we will define:

  • xi1x_{i1}xi1​ as the weight (wt) of the iiith car.
  • xi2x_{i2}xi2​ as the model year (year) of the iiith car.

How do we find such a plane? Well, we would like a plane that is as close as possible to the data points. That is, we would like it to minimize the errors it is making. How will we define these errors? Squared distance of course! So, we would like to minimize

f(β0,β1,β2)=∑i=1n(yi−(β0+β1xi1+β2xi2))2f(\beta_0, \beta_1, \beta_2) = \sum_{i = 1}^{n}(y_i - (\beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2}))^2f(β0​,β1​,β2​)=i=1∑n​(yi​−(β0​+β1​xi1​+β2​xi2​))2

with respect to β0\beta_0β0​, β1\beta_1β1​, and β2\beta_2β2​. How do we do so? It is another straightforward multivariate calculus problem. All we have done is add an extra variable since we did this last time. So again, we take a derivative with respect to each of β0\beta_0β0​, β1\beta_1β1​, and β2\beta_2β2​ and set them equal to zero, then solve the resulting system of equations. That is,

∂f∂β0=0∂f∂β1=0∂f∂β2=0\begin{aligned} \frac{\partial f}{\partial \beta_0} &= 0 \\ \frac{\partial f}{\partial \beta_1} &= 0 \\ \frac{\partial f}{\partial \beta_2} &= 0 \end{aligned}∂β0​∂f​∂β1​∂f​∂β2​∂f​​=0=0=0​

After doing so, we will once again obtain the normal equations.

nβ0+β1∑i=1nxi1+β2∑i=1nxi2=∑i=1nyiβ0∑i=1nxi1+β1∑i=1nxi12+β2∑i=1nxi1xi2=∑i=1nxi1yiβ0∑i=1nxi2+β1∑i=1nxi1xi2+β2∑i=1nxi22=∑i=1nxi2yi\begin{aligned} n \beta_0 + \beta_1 \sum_{i = 1}^{n} x_{i1} + \beta_2 \sum_{i = 1}^{n} x_{i2} &= \sum_{i = 1}^{n} y_i \\ \beta_0 \sum_{i = 1}^{n} x_{i1} + \beta_1 \sum_{i = 1}^{n} x_{i1}^2 + \beta_2 \sum_{i = 1}^{n} x_{i1}x_{i2} &= \sum_{i = 1}^{n} x_{i1}y_i \\ \beta_0 \sum_{i = 1}^{n} x_{i2} + \beta_1 \sum_{i = 1}^{n} x_{i1}x_{i2} + \beta_2 \sum_{i = 1}^{n} x_{i2}^2 &= \sum_{i = 1}^{n} x_{i2}y_i \end{aligned}nβ0​+β1​i=1∑n​xi1​+β2​i=1∑n​xi2​β0​i=1∑n​xi1​+β1​i=1∑n​xi12​+β2​i=1∑n​xi1​xi2​β0​i=1∑n​xi2​+β1​i=1∑n​xi1​xi2​+β2​i=1∑n​xi22​​=i=1∑n​yi​=i=1∑n​xi1​yi​=i=1∑n​xi2​yi​​

We now have three equations and three variables, which we could solve, or we could simply let R solve for us.

coef(mpg_model)

Here we have once again fit our model using lm(), however we have introduced a new syntactical element. The formula mpg ~ wt + year now reads: "model the response variable mpg as a linear function of wt and year". That is, it will estimate an intercept, as well as slope coefficients for wt and year. We then extract these as we have done before using coef().

In the multiple linear regression setting, some of the interpretations of the coefficients change slightly.

Here, β^0\hat{\beta}_0β^​0​ is our estimate for β0\beta_0β0​, the mean miles per gallon for a car that weighs 0 pounds and was built in 1900. We see our estimate here is negative, which is a physical impossibility. However, this isn't unexpected, as we shouldn't expect our model to be accurate for cars from 1900 which weigh 0 pounds. (Because they never existed!) This isn't much of a change from SLR. That is, β0\beta_0β0​ is still simply the mean when all of the predictors are 0.

Coefficients and Interpretation

Criticallty, the interpretation of the coefficients in front of our predictors are slightly different than before.

For example β^1=\hat{\beta}_1 =β^​1​= is our estimate for β1\beta_1β1​, the average change in miles per gallon for an increase in weight (x1x_{1}x1​) of one-pound for a car of a certain model year. That is, we hold all other variables constant. Here, this means β^1\hat{\beta}_1β^​1​ should be interpreted for a fixed value of x2x_{2}x2​. Note that this coefficient is actually the same for any given value of x2x_{2}x2​.

Later, we will look at models that allow for a different change in mean response for different values of x2x_{2}x2​. Also note that this estimate is negative, which we would expect since, in general, fuel efficiency decreases for larger vehicles. Note that in the multiple linear regression setting, this interpretation is dependent on a fixed value for x2x_{2}x2​---that is, "for a car of a certain model year." It is possible that the indirect relationship between fuel efficiency and weight does not hold when an additional factor, say year, is included, and thus we could have the sign of our coefficient flipped.

Lastly, β^2=\hat{\beta}_2 =β^​2​= is our estimate for β2\beta_2β2​, the average change in miles per gallon for a one-year increase in model year (x2x_{2}x2​) for a car of a certain weight, that is, for a fixed value of x1x_{1}x1​. It is not surprising that the estimate is positive. We expect that as time passes and the years march on, technology would improve so that a car of a specific weight would get better mileage now as compared to their predecessors. And yet, the coefficient could have been negative because we are also including weight as variable, and not strictly as a fixed value.

Matrix Approach to Regression

In our above example we used two predictor variables, but it will only take a little more work to allow for an arbitrary number of predictor variables and derive their coefficient estimates. We can consider the model,

Yi=β0+β1xi1+β2xi2+⋯+βp−1xi(p−1)+ϵi,i=1,2,…,nY_i = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \cdots + \beta_{p-1} x_{i(p-1)} + \epsilon_i, \qquad i = 1, 2, \ldots, nYi​=β0​+β1​xi1​+β2​xi2​+⋯+βp−1​xi(p−1)​+ϵi​,i=1,2,…,n

where ϵi∼N(0,σ2)\epsilon_i \sim N(0, \sigma^2)ϵi​∼N(0,σ2). In this model, there are p−1p - 1p−1 predictor variables, x1,x2,⋯ ,xp−1x_1, x_2, \cdots, x_{p-1}x1​,x2​,⋯,xp−1​. There are a total of ppp β\betaβ-parameters and a single parameter σ2\sigma^2σ2 for the variance of the errors. (It should be noted that almost as often, authors will use ppp as the number of predictors, making the total number of β\betaβ parameters p+1p+1p+1. This is always something you should be aware of when reading about multiple regression. There is not a standard that is used most often.)

If we were to stack together the nnn linear equations that represent each YiY_iYi​ into a column vector, we get the following.

Regression Solution

The normal equations can be written much more succinctly in matrix notation,

X⊤Xβ=X⊤y.X^\top X \beta = X^\top y.X⊤Xβ=X⊤y.

We can then solve this expression by multiplying both sides by the inverse of X⊤XX^\top XX⊤X, which exists, provided the columns of XXX are linearly independent. Then as always, we denote our solution with a hat.

β^=(X⊤X)−1X⊤y\hat{\beta} = \left( X^\top X \right)^{-1}X^\top yβ^​=(X⊤X)−1X⊤y

To verify that this is what R has done for us in the case of two predictors, we create an XXX matrix. Note that the first column is all 1s, and the remaining columns contain the data.

n = nrow(autompg)
p = length(coef(mpg_model))
X = cbind(rep(1, n), autompg$wt, autompg$year)
y = autompg$mpg

(beta_hat = solve(t(X) %*% X) %*% t(X) %*% y)
coef(mpg_model)

β^=[−14.6−0.00660.76]\hat{\beta} = \begin{bmatrix} −14.6 \\ −0.0066 \\ 0.76 \\ \end{bmatrix}β^​=⎣⎢⎡​−14.6−0.00660.76​⎦⎥⎤​

In our new notation, the fitted values can be written

y^=Xβ^.\hat{y} = X \hat{\beta}.y^​=Xβ^​.

y^=[y^1y^2⋮y^n]\hat{y} = \begin{bmatrix} \hat{y}_1 \\ \hat{y}_2 \\ \vdots\\ \hat{y}_n \end{bmatrix}y^​=⎣⎢⎢⎢⎢⎡​y^​1​y^​2​⋮y^​n​​⎦⎥⎥⎥⎥⎤​

Then, we can create a vector for the residual values,

e=[e1e2⋮en]=[y1y2⋮yn]−[y^1y^2⋮y^n].e = \begin{bmatrix} e_1 \\ e_2 \\ \vdots\\ e_n \end{bmatrix} = \begin{bmatrix} y_1 \\ y_2 \\ \vdots\\ y_n \end{bmatrix} - \begin{bmatrix} \hat{y}_1 \\ \hat{y}_2 \\ \vdots\\ \hat{y}_n \end{bmatrix}.e=⎣⎢⎢⎢⎢⎡​e1​e2​⋮en​​⎦⎥⎥⎥⎥⎤​=⎣⎢⎢⎢⎢⎡​y1​y2​⋮yn​​⎦⎥⎥⎥⎥⎤​−⎣⎢⎢⎢⎢⎡​y^​1​y^​2​⋮y^​n​​⎦⎥⎥⎥⎥⎤​.

And lastly, we can update our estimate for σ2\sigma^2σ2.

se2=∑i=1n(yi−y^i)2n−p=e⊤en−ps_e^2 = \frac{\sum_{i=1}^n (y_i - \hat{y}_i)^2}{n - p} = \frac{e^\top e}{n-p}se2​=n−p∑i=1n​(yi​−y^​i​)2​=n−pe⊤e​

Sampling Distribution

This subsection includes technical notes. Skip Ahead

As we can see in the output below, the results of calling summary() are similar to simple linear regression, but there are some differences, most obviously a new row for the added predictor variable.

summary(mpg_model)

To understand these differences in detail, we will need to first obtain the sampling distribution of β^\hat{\beta}β^​.

The derivation of the sampling distribution of β^\hat{\beta}β^​ involves the multivariate normal distribution. A full discussion of this distribution is outside the scope of this course.

Our goal now is to obtain the distribution of the β^\hat{\beta}β^​ vector,

β^=[β^0β^1β^2⋮β^p−1]\hat{\beta} = \begin{bmatrix} \hat{\beta}_0 \\ \hat{\beta}_1 \\ \hat{\beta}_2 \\ \vdots \\ \hat{\beta}_{p-1} \end{bmatrix}β^​=⎣⎢⎢⎢⎢⎢⎢⎢⎡​β^​0​β^​1​β^​2​⋮β^​p−1​​⎦⎥⎥⎥⎥⎥⎥⎥⎤​

Recall from last time that when discussing sampling distributions, we now consider β^\hat{\beta}β^​ to be a random vector, thus we use YYY instead of the data vector yyy.

β^=(X⊤X)−1X⊤Y\hat{\beta} = \left( X^\top X \right)^{-1}X^\top Yβ^​=(X⊤X)−1X⊤Y

Then it is a consequence of the multivariate normal distribution that,

β^∼N(β,σ2(X⊤X)−1).\hat{\beta} \sim N\left(\beta, \sigma^2 \left(X^\top X\right)^{-1} \right).β^​∼N(β,σ2(X⊤X)−1).

We then have

E[β^]=β\text{E}[\hat{\beta}] = \betaE[β^​]=β

and for any β^j\hat{\beta}_jβ^​j​ we have

E[β^j]=βj.\text{E}[\hat{\beta}_j] = \beta_j.E[β^​j​]=βj​.

We also have

Var[β^]=σ2(X⊤X)−1\text{Var}[\hat{\beta}] = \sigma^2 \left( X^\top X \right)^{-1}Var[β^​]=σ2(X⊤X)−1

and for any β^j\hat{\beta}_jβ^​j​ we have

Var[β^j]=σ2Cjj\text{Var}[\hat{\beta}_j] = \sigma^2 C_{jj}Var[β^​j​]=σ2Cjj​

where

C=(X⊤X)−1C = \left(X^\top X\right)^{-1}C=(X⊤X)−1

and the elements of CCC are denoted

Essentially, the diagonal elements correspond to the β\betaβ vector.

The standard error for the β^\hat{\beta}β^​ vector is given by

SE[β^]=se(X⊤X)−1\text{SE}[\hat{\beta}] = s_e \sqrt{\left( X^\top X \right)^{-1}}SE[β^​]=se​(X⊤X)−1​

and for a particular β^j\hat{\beta}_jβ^​j​

SE[β^j]=seCjj.\text{SE}[\hat{\beta}_j] = s_e \sqrt{C_{jj}}.SE[β^​j​]=se​Cjj​​.

Lastly, each of the β^j\hat{\beta}_jβ^​j​ follows a normal distribution,

β^j∼N(βj,σ2Cjj).\hat{\beta}_j \sim N\left(\beta_j, \sigma^2 C_{jj} \right).β^​j​∼N(βj​,σ2Cjj​).

thus

β^j−βjseCjj∼tn−p.\frac{\hat{\beta}_j - \beta_j}{s_e \sqrt{C_{jj}}} \sim t_{n-p}.se​Cjj​​β^​j​−βj​​∼tn−p​.

Now that we have the necessary distributional results, we can move on to perform tests and make interval estimates.

Single Parameter Tests

The first test we will see is a test for a single βj\beta_jβj​.

H0:βj=0vsH1:βj≠0H_0: \beta_j = 0 \quad \text{vs} \quad H_1: \beta_j \neq 0H0​:βj​=0vsH1​:βj​​=0

Again, the test statistic takes the form

TS=EST−HYPSE.\text{TS} = \frac{\text{EST} - \text{HYP}}{\text{SE}}.TS=SEEST−HYP​.

In particular,

t=β^j−βjSE[β^j]=β^j−0seCjj,t = \frac{\hat{\beta}_j - \beta_j}{\text{SE}[\hat{\beta}_j]} = \frac{\hat{\beta}_j-0}{s_e\sqrt{C_{jj}}},t=SE[β^​j​]β^​j​−βj​​=se​Cjj​​β^​j​−0​,

which, under the null hypothesis, follows a ttt distribution with n−pn - pn−p degrees of freedom.

Recall our model for mpg,

where ϵi∼N(0,σ2)\epsilon_i \sim N(0, \sigma^2)ϵi​∼N(0,σ2).

Confidence Intervals for Mean Response

As we saw in SLR, we can create confidence intervals for the mean response, that is, an interval estimate for E[Y∣X=x]\text{E}[Y \mid X = x]E[Y∣X=x]. In SLR, the mean of YYY was only dependent on a single value xxx. Now, in multiple regression, E[Y∣X=x]\text{E}[Y \mid X = x]E[Y∣X=x] is dependent on the value of each of the predictors, so we define the vector x0x_0x0​ to be,

x0=[1x01x02⋮x0(p−1)].x_{0} = \begin{bmatrix} 1 \\ x_{01} \\ x_{02} \\ \vdots \\ x_{0(p-1)} \\ \end{bmatrix}.x0​=⎣⎢⎢⎢⎢⎢⎢⎡​1x01​x02​⋮x0(p−1)​​⎦⎥⎥⎥⎥⎥⎥⎤​.

y^(x0)=x0⊤β^=β^0+β^1x01+β^2x02+⋯+β^p−1x0(p−1).\begin{aligned} \hat{y}(x_0) &= x_{0}^\top\hat{\beta} \\ &= \hat{\beta}_0 + \hat{\beta}_1 x_{01} + \hat{\beta}_2 x_{02} + \cdots + \hat{\beta}_{p-1} x_{0(p-1)}. \end{aligned}y^​(x0​)​=x0⊤​β^​=β^​0​+β^​1​x01​+β^​2​x02​+⋯+β^​p−1​x0(p−1)​.​

E[y^(x0)]=x0⊤β=β0+β1x01+β2x02+⋯+βp−1x0(p−1)\begin{aligned} \text{E}[\hat{y}(x_0)] &= x_{0}^\top\beta \\ &= \beta_0 + \beta_1 x_{01} + \beta_2 x_{02} + \cdots + \beta_{p-1} x_{0(p-1)} \end{aligned}E[y^​(x0​)]​=x0⊤​β=β0​+β1​x01​+β2​x02​+⋯+βp−1​x0(p−1)​​

Significance of Regression

The decomposition of variation that we had seen in SLR still holds for MLR.

∑i=1n(yi−yˉ)2=∑i=1n(yi−y^i)2+∑i=1n(y^i−yˉ)2\sum_{i=1}^{n}(y_i - \bar{y})^2 = \sum_{i=1}^{n}(y_i - \hat{y}_i)^2 + \sum_{i=1}^{n}(\hat{y}_i - \bar{y})^2i=1∑n​(yi​−yˉ​)2=i=1∑n​(yi​−y^​i​)2+i=1∑n​(y^​i​−yˉ​)2

That is,

SST=SSE+SSReg.\text{SST} = \text{SSE} + \text{SSReg}.SST=SSE+SSReg.

This means that, we can calculate R2R^2R2 in a simple manner. R continues to do this automatically.

summary(mpg_model)$r.squared

The interpretation changes slightly as compared to SLR. As an example, we would say that R2R^2R2 percent of the observed variation in miles per gallon is explained by the linear relationship with the two predictor variables, weight and year.

In multiple regression, a natural test is the following:

H0:β1=β2=⋯=βp−1=0.H_0: \beta_1 = \beta_2 = \cdots = \beta_{p - 1} = 0.H0​:β1​=β2​=⋯=βp−1​=0.

Here, we see that the null hypothesis sets all of the βj\beta_jβj​ equal to 0, except the intercept, β0\beta_0β0​. We could then say that the null model, or "model under the null hypothesis" is

Yi=β0+ϵi.Y_i = \beta_0 + \epsilon_i.Yi​=β0​+ϵi​.

This is a model where the regression is insignificant. None of the predictors have a significant linear relationship with the response. Notationally, we will denote the fitted values of this model as y^0i\hat{y}_{0i}y^​0i​, which in this case happens to be:

y^0i=yˉ.\hat{y}_{0i} = \bar{y}.y^​0i​=yˉ​.

The alternative hypothesis here is that at least one of the βj\beta_jβj​ from the null hypothesis is not 0.

H1:At least one of βj≠0,j=1,2,⋯ ,(p−1)H_1: \text{At least one of } \beta_j \neq 0, j = 1, 2, \cdots, (p-1)H1​:At least one of βj​​=0,j=1,2,⋯,(p−1)

We could then say that the full model, or "model under the alternative hypothesis" is

Yi=β0+β1xi1+β2xi2+⋯+β(p−1)xi(p−1)+ϵiY_i = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \cdots + \beta_{(p-1)} x_{i(p-1)} + \epsilon_iYi​=β0​+β1​xi1​+β2​xi2​+⋯+β(p−1)​xi(p−1)​+ϵi​

This is a model where the regression is significant. At least one of the predictors has a significant linear relationship with the response. There is some linear relationship between yyy and the predictors, x1,x2,…,xp−1x_1, x_2, \ldots, x_{p - 1}x1​,x2​,…,xp−1​.

We will denote the fitted values of this model as y^1i\hat{y}_{1i}y^​1i​.

To develop the FFF test for the significance of the regression, we will arrange the variance decomposition into an ANOVA table.

In summary, the FFF statistic is

F=∑i=1n(y^1i−yˉ)2/(p−1)∑i=1n(yi−y^1i)2/(n−p),F = \frac{\sum_{i=1}^{n}(\hat{y}_{1i} - \bar{y})^2 / (p - 1)}{\sum_{i=1}^{n}(y_i - \hat{y}_{1i})^2 / (n - p)},F=∑i=1n​(yi​−y^​1i​)2/(n−p)∑i=1n​(y^​1i​−yˉ​)2/(p−1)​,

since we reject for large values of FFF. A large value of the statistic corresponds to a large portion of the variance being explained by the regression. Here Fp−1,n−pF_{p-1, n-p}Fp−1,n−p​ represents a random variable which follows an FFF distribution with p−1p - 1p−1 and n−pn - pn−p degrees of freedom.

To perform this test in R, we first explicitly specify the two models in R and save the results in different variables. We then use anova() to compare the two models, giving anova() the null model first and the alternative (full) model second. (Specifying the full model first will result in the same p-value, but some nonsensical intermediate values.)

In this case,

  • H0H_0H0​: Yi=β0+ϵiY_i = \beta_0 + \epsilon_iYi​=β0​+ϵi​

That is, in the null model, we use neither of the predictors, whereas in the full (alternative) model, at least one of the predictors is useful.

Nested Models

The significance of regression test is actually a special case of testing what we will call nested models. This is a powerful tool. Using it we can compare two models, where one model is "nested" inside the other. This means one model contains a subset of the predictors from only the larger model.

Consider the following full model,

Yi=β0+β1xi1+β2xi2+⋯+β(p−1)xi(p−1)+ϵiY_i = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \cdots + \beta_{(p-1)} x_{i(p-1)} + \epsilon_iYi​=β0​+β1​xi1​+β2​xi2​+⋯+β(p−1)​xi(p−1)​+ϵi​

This model has p−1p - 1p−1 predictors, for a total of ppp β\betaβ-parameters. We will denote the fitted values of this model as y^1i\hat{y}_{1i}y^​1i​.

Let the null model be

Yi=β0+β1xi1+β2xi2+⋯+β(q−1)xi(q−1)+ϵiY_i = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \cdots + \beta_{(q-1)} x_{i(q-1)} + \epsilon_iYi​=β0​+β1​xi1​+β2​xi2​+⋯+β(q−1)​xi(q−1)​+ϵi​

where q<pq < pq<p. This model has q−1q - 1q−1 predictors, for a total of qqq β\betaβ-parameters. We will denote the fitted values of this model as y^0i\hat{y}_{0i}y^​0i​.

The difference between these two models can be codified by the null hypothesis of a test.

H0:βq=βq+1=⋯=βp−1=0.H_0: \beta_q = \beta_{q+1} = \cdots = \beta_{p - 1} = 0.H0​:βq​=βq+1​=⋯=βp−1​=0.

Specifically, the β\betaβ-parameters from the full model that are not in the null model are zero. The resulting model, which is nested, is the null model.

We can then perform this test using an FFF-test, which is the result of the following ANOVA table.

The alternative is simply that at least one of the βj\beta_{j}βj​ from the null is not 0.

To perform this test in R we first define both models, then give them to the anova() commands.

null_mpg_model = lm(mpg ~ wt + year, data = autompg)
#full_mpg_model = lm(mpg ~ wt + year + cyl + disp + hp + acc, data = autompg)
full_mpg_model = lm(mpg ~ ., data = autompg)
anova(null_mpg_model, full_mpg_model)

Here we have used the formula mpg ~ . to define to full model. This is the same as the commented out line. Specifically, this is a common shortcut in R which reads, "model mpg as the response with each of the remaining variables in the data frame as predictors."

Simulation

Since we ignored the derivation of certain results, we will again use simulation to convince ourselves of some of the above results. In particular, we will simulate samples of size n = 100 from the model

set.seed(1337)
n = 100 # sample size
p = 3

beta_0 = 5
beta_1 = -2
beta_2 = 6
sigma  = 4

As is the norm with regression, the xxx values are considered fixed and known quantities, so we will simulate those first, and they remain the same for the rest of the simulation study. Also note we create an x0 which is all 1, which we need to create our X matrix. If you look at the matrix formulation of regression, this unit vector of all 1s is a "predictor" that puts the intercept into the model. We also calculate the C matrix for later use.

x0 = rep(1, n)
x1 = sample(seq(1, 10, length = n))
x2 = sample(seq(1, 10, length = n))
X = cbind(x0, x1, x2)
C = solve(t(X) %*% X)

We then simulate the response according the model above. Lastly, we place the two predictors and response into a data frame. Note that we do not place x0 in the data frame. This is a result of R adding an intercept by default.

eps      = rnorm(n, mean = 0, sd = sigma)
y        = beta_0 + beta_1 * x1 + beta_2 * x2 + eps
sim_data = data.frame(x1, x2, y)

So far so good. Everything checks out. Now we will finally simulate from this model repeatedly in order to obtain an empirical distribution of β^2\hat{\beta}_2β^​2​.

We expect β^2\hat{\beta}_2β^​2​ to follow a normal distribution,

β^2∼N(β2,σ2C22).\hat{\beta}_2 \sim N\left(\beta_2, \sigma^2 C_{22} \right).β^​2​∼N(β2​,σ2C22​).

We now perform the simulation a large number of times. Each time, we update the y variable in the data frame, leaving the x variables the same. We then fit a model, and store β^2\hat{\beta}_2β^​2​.

num_sims = 10000
beta_hat_2 = rep(0, num_sims)
for(i in 1:num_sims) {
  eps           = rnorm(n, mean = 0 , sd = sigma)
  sim_data$y    = beta_0 * x0 + beta_1 * x1 + beta_2 * x2 + eps
  fit           = lm(y ~ x1 + x2, data = sim_data)
  beta_hat_2[i] = coef(fit)[3]
}
